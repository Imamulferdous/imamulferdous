## 🔧 Renaming Rules Applied

1. **All lowercase** - easier to manage
2. **Hyphens instead of spaces** - web-friendly
3. **No special characters** - only letters, numbers, hyphens
4. **Descriptive** - title-based for easy identification
5. **No abbreviations** - full words for clarity

---

## 💡 How to Rename on Different Systems

### Windows:
1. Right-click the PDF file
2. Select "Rename"
3. Copy the new name from this guide
4. Paste and press Enter

### Mac:
1. Click on the file once
2. Press Enter/Return
3. Copy the new name from this guide
4. Paste and press Enter

### Linux:
```bash
mv "original-name.pdf" "new-name-from-guide.pdf"
```
